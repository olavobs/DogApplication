# DogApplication
DogApplication using Springboot and h2 database with JPA based on Java Developer Nanodegree from Udacity.


## Start Application: 

- Application will run on port 8081

### Endpoints:
  - /dogs
  - /dogs/breed
  - /{id}/breed
  - /dogs/name
